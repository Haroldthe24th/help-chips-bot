Chipsbot tracks the toxicity of users.

This is a help page, the following can also be accases using c!help in a discord server with the Chipsbot in it.

Chips Bot

Command Usage
c!toxic "message" returns the toxicity of the message.
c!mosttoxic eturns a list of the most toxic users on the server.
c!mosttoxicmessages(c!leaderboard) returns the highest scoring messages.
c!averagetoxicity(c!average) tells you how toxic a user is on average
c!toxicpoints(c!total) returns toxicity points.
c!summary(c!userstats) gives you the users message count, toxicity score, and average toxicity.
c!feedback "your message" Gives The devs feedback
