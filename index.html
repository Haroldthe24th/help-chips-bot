This is a help page. Use the c!help command.


Command Description Example Usage
c!toxic [@user] Determines how toxic your message is based on a machine learning algorithm. Click the "made using" link to learn more c!toxic your mom is a fat cow
> Toxicity Rating: 78.55312%
c!top Prints a list of the top 10 most toxic users in this server c!top
c!leaderboard Prints the most toxic messages of all time. Use c!toxic to try for the leaderboard. c!leaderboard
c!average[@user] Tells you this user's average toxicity coefficient is based on their post history. c!average @chips
> @chips has an average toxicity coefficient of 30.23%
c!total [@user] Returns the total toxicity point accumulation for a user c!total @chips
> @chips has a cumulative toxicity score of 114.20 points
c!userstats [@user] gives you the users message count, toxicity score, and average toxicity. c!userstats @chips
> @chips has 379 posts and 114.82 toxicity points. Met this user on 2018-09-01
c!feedback [message] Leave feedback for the developer c!feedback You're a loser and your bot sucks
> Your message will be read, thank you!
c!cuck [@user]or [message] Determines once and for all whether or not someone is a cuckold c!cuck Donald Trump
